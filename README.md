# 🛒 SmartShelfX – AI-Powered Inventory Management System

## 📌 Overview

SmartShelfX is a full-stack inventory management system that helps businesses efficiently manage stock, monitor inventory levels, analyze product data, and predict future demand using machine learning techniques.

The platform provides inventory tracking, demand forecasting, low-stock alerts, role-based access control, and real-time analytics to improve inventory decision-making.

---

## 🚀 Features

### 📦 Inventory Management

* Add, update, delete, and view products
* Stock quantity tracking
* Category-wise product management

### 🔐 Authentication & Authorization

* Secure Login & Registration
* JWT-based Authentication
* Role-Based Access Control

### 📊 Analytics Dashboard

* Inventory statistics
* Product insights
* Sales and stock reports

### 🤖 AI Demand Forecasting

* Predict future product demand
* Machine Learning-based forecasting
* Data-driven inventory planning

### 🚨 Stock Alerts

* Low-stock notifications
* Reorder recommendations

### 👥 User Roles

* Admin
* Manager
* Vendor/User

## ⚙️ Installation & Setup

### 1. Clone Repository

git clone https://github.com/AkitiShravani/SmartShelfX.git

### 2. Install Frontend Dependencies

cd frontend
npm install

### 3. Install Backend Dependencies

cd backend
npm install

### 4. Install ML Dependencies

cd ml-service
pip install -r requirements.txt

---

## ▶️ Running the Project

### Start Backend

cd backend
node server.js

### Start Frontend

cd frontend
ng serve

### Start ML Service

cd ml-service
python main.py
